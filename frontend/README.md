# 可京室內裝修 - 前端

室內裝修服務網站的Next.js前端部分

## 技術棧

- Next.js 14
- React
- TypeScript
- Tailwind CSS
- Shadcn UI 組件

## 開發環境設置

```bash
# 安裝依賴
npm install

# 運行開發服務器
npm run dev

# 構建生產版本
npm run build

# 啟動生產服務器
npm run start
```

## 目錄結構

- `src/app/` - Next.js App Router 頁面和路由
- `src/components/` - React 組件
- `public/` - 靜態資源
- `src/app/globals.css` - 全局樣式

## 特點

- 響應式設計，支持各種設備尺寸
- 現代化UI界面
- 案例展示
- 服務介紹
- 專業FAQ問答
